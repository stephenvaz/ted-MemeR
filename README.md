# A random Meme Generator built in React
APIs:
- [MemeAPI](https://github.com/D3vd/Meme_Api)
- [ShortURL](https://rapidapi.com/Walter678/api/shorturl9)

[Deployed Website](https://ted-meme-r.vercel.app/)

## Steps to run:
### Development
- npm install
- npm run dev
### Production
- npm run build
- npm run preview
